Data SelectionThe study utilizes the Industrial Production Index (IPI) for the Manufacture of Dairy Products (Identifiant INSEE: 010767631).The series is corrected for seasonal variations and working days (CVS-CJO).It consists of monthly observations (frequency 12) with a sample size exceeding 100 observations.The analysis includes data processing, such as logarithmic transformations and differentiation, to ensure stationarity.MethodologyThe project is divided into three core analytical parts:Part I: Data Preparation & Stationarity * Descriptive analysis and graphical representation of the series.Implementation of transformations (differentiation, trend correction) to achieve a stationary process.Part II: ARMA Modeling * Selection of an appropriate $ARMA(p, q)$ model based on statistical criteria.Parameter estimation and diagnostic checking to ensure model validity.Formal specification of the resulting $ARIMA(p, d, q)$ process.Part III: Prediction * Derivation of the equation for the 95% confidence region for future values $(X_{T+1}, X_{T+2})$.Analysis of predictive improvements using auxiliary stationary time series.
